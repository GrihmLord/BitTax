1. **Clone the Repository**

    ```bash
    git clone https://github.com/yourusername/BitTax.git
    cd BitTax
    ```

2. **Install Dependencies**

    ```bash
    npm install
    ```

    ```bash
    yarn install
    ```

3. **Run the App**

    ```bash

    ```

    ```bash
    npm start
    ```

    or

    ```bash
    yarn start
    ```
